# Ayureva Platform

This is the monorepo for the Ayureva platform, containing both the web application (Next.js) and the mobile application (Flutter).

## Structure

*   **/web**: Next.js application (Admin Panel, Landing Page, Backend API).
*   **/mobile**: Flutter application (Android/iOS for Students).

## Getting Started

### Web
1.  `cd web`
2.  `npm install`
3.  `npm run dev`

### Mobile
1.  `cd mobile`
2.  `flutter pub get`
3.  `flutter run`
